# Multimodal AI Gaming Agent (CS2 Assistant)

Универсальный модульный ИИ-ассистент, интегрированный в систему для взаимодействия с игровым окружением в реальном времени. Проект сочетает в себе компьютерное зрение (OCR), распознавание речи (STT), локальные языковые модели (LLM) и синтез речи (TTS) для создания полноценного интерактивного агента.

## 🚀 Ключевые особенности

* **Real-time Screen Perception:** Использование библиотеки `DXcam` для высокоскоростного захвата экрана с минимальными задержками.
* **Local-First Inference:** Полностью локальный запуск всех компонентов (Whisper, Ollama, QwenTTS), что обеспечивает приватность и нулевой latency от сети.
* **Async Pipeline:** Весь цикл — от захвата аудио до генерации ответа — построен на `asyncio` с выносом тяжелых вычислений в отдельные потоки.
* **System Integration:** Ассистент имеет доступ к управлению приложениями и системными режимами через механизм Tool Calling.
* **Adaptive Model Management:** Динамическое переключение LLM и TTS моделей с автоматической выгрузкой неиспользуемых моделей из VRAM.

## 🛠 Инструментарий и System Interaction (Tool Calling)

Одной из ключевых особенностей проекта является способность LLM вызывать программные инструменты для выполнения задач в ОС. Текущая реализация `registry.py` включает:

* **Управление играми:** Команда "Запусти КС" или "Давай играть" инициирует запуск CS2 напрямую через Steam протокол (`steam://rungameid/730`).
* **Динамическая смена состояний (Game Mode):** По команде "Активируй игровой режим" ассистент может переключать используемую модель (с тяжелой локальной Ollama на облачную Gemini) и менять системный промпт на более агрессивный/игровой.
* **Мультимедиа:** Интеграция с YouTube — ассистент может найти и запустить видео или музыку по голосовому запросу, используя парсинг поисковой выдачи и автоматическое открытие браузера.

## 💻 Технологический стек

* **Core:** Python 3.10+, Asyncio
* **ML/AI:**  
    * `Faster-Whisper` (STT) — распознавание речи  
    * `Ollama` / `Gemini` (LLM) — логическое ядро ассистента  
    * `Faster-Qwen3-TTS` / `Silero V5` (TTS) — синтез речи (качество / скорость)  
* **Computer Vision:** `DXcam`, `OpenCV`
* **DevOps & Config:** `Pydantic Settings` (v2), `YAML`

## 🏗 Архитектура

Проект спроектирован как расширяемая система агентов:

1. **Audio Recorder:** Захват микрофона по нажатию (PTT)  
2. **Multimodal Input:** Параллельный захват экрана и транскрибация речи  
3. **LLM Manager:** Оркестрация моделей, управление историей диалога, Tool Calling и lifecycle моделей (загрузка/выгрузка из VRAM)  
4. **TTS Manager:** Динамическое управление моделями синтеза речи (переключение режимов, выгрузка из VRAM)
